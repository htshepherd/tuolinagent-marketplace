# tuolinagent 极简安装手册

打开 Codex，切换到 Plan 模式，按下面顺序复制。

## 1. 安装

```text
请从这个 Codex 插件市场仓库安装 tuolinagent：
https://github.com/htshepherd/tuolinagent-marketplace

请使用下面两条命令：
codex plugin marketplace add htshepherd/tuolinagent-marketplace
codex plugin add tuolinagent@tuolinagent-marketplace

请只使用上面的安装地址和命令。

安装前先检查我的 Windows 电脑依赖。请运行仓库里的 scripts/windows_check_dependencies.ps1。

如果缺少 Git、Python 3.10+、ffmpeg 或 MinerU，请先告诉我缺少什么、为什么需要、准备执行什么安装命令。

在我回复“同意安装依赖”之前，不要安装任何软件。
```

## 2. 同意安装依赖

Codex 列出缺少项后，复制：

```text
同意安装依赖。只安装刚才列出的缺少项，安装后重新检查。
```

如果只缺少 Graphify，复制：

```text
Graphify 先不安装。默认 codex_adapter 模式不需要 Graphify API key。
```

## 3. 配置 raw 目录

把路径换成你的真实 raw 路径：

```text
请在当前项目里创建 config/tuolin-kb.config.json。
我的 raw 目录是 D:\LocalKnowledge\raw，请在配置里写成 D:/LocalKnowledge/raw。
保持 codex_adapter 模式，不要要求 Graphify API key。
不要移动、删除或重命名我的 raw 目录。
```

## 4. 检查配置

```text
检查一下知识库配置是否可用。
```

## 5. 常用自然语言命令

查看还要做什么：

```text
查看当前还有哪些分区需要继续整理。
```

让 Codex 推荐下一步：

```text
整理一下拓霖知识库。
请推荐一个最有价值的下一步，并给我一条可以直接复制的确认回复。
执行前等我确认。
```

按推荐继续：

```text
确认，按推荐的下一步执行。
```

继续看某个分区：

```text
确认，继续看标准资料。
```

```text
确认，继续看市场资料。
```

把已识别内容整理成可用资料：

```text
确认，先把【分区名称】整理成可用资料。
```

更新知识库：

```text
更新一下知识库。
请先检查 raw 目录有哪些变化。
按构建分区更新，不要全量重建。
执行前等我确认。
```

问答测试：

```text
根据当前知识库回答：【这里填写问题】
如果知识库里没有确定答案，请直接说没有确定资料，不要编。
```

查看复核内容：

```text
有哪些内容需要我复核？
```

## 6. 开始整理

```text
整理一下知识库。
请按构建分区来做，不要全量扫描所有原始资料。
先推荐一个最有价值的下一步，执行前等我确认。
```

## 7. 确认执行

```text
确认，按推荐的下一步执行。
```

## 8. 查看状态

```text
查看知识库状态。
```

```text
查看当前有哪些构建分区，哪些已经可用，哪些还需要继续整理。
```

## 9. 整理知识库

```text
整理一下知识库。
请根据当前资料状态推荐一个最有价值的下一步。
执行前等我确认。
```

```text
整理成可用资料。
请把已经识别出的内容整理进对应知识包，需要我判断的内容单独列出来。
执行前等我确认。
```

```text
继续看资料。
请按当前推荐的构建分区继续，不要跨分区处理。
执行前等我确认。
```

## 10. 更新知识库

```text
更新一下知识库。
请先检查 raw 目录有哪些变化。
按构建分区更新，不要全量重建。
执行前等我确认。
```

```text
我刚刚往 raw 目录补充了新资料。
请检查哪些构建分区需要更新，并推荐下一步。
执行前等我确认。
```

## 11. 指定分区整理

把名称换成你要整理的产品或知识域：

```text
整理一下【分区名称】资料。
只处理这个构建分区，不要处理其他分区。
执行前等我确认。
```

```text
继续看【分区名称】剩下的资料。
只处理这个构建分区。
执行前等我确认。
```

## 12. 问答测试

```text

如果知识库里没有确定答案，请直接说没有确定资料，不要编。
```

```text
【这里填写产品或主题】现在有哪些可用资料？
```

```text
【这里填写产品或主题】适合哪些场景？
```

```text
【这里填写产品或主题】有哪些内容还需要我确认？
```

## 13. 复核测试

```text
有哪些内容需要我复核？
```

```text
请展示第 1 条复核内容的来源和建议处理方式。
不要写回核心资料，先让我确认。
```

```text
这条复核通过。

```

```text
请告诉我 graph.html 在哪里。
我要打开本地知识图谱查看节点关系。
```

```text

```

## 15. 停止或重新推荐

```text
先不要执行。
请重新推荐下一步，并说明为什么。
```

```text
暂停整理。
只告诉我当前状态和下一步建议。
```
