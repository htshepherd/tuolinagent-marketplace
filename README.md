# tuolinagent marketplace

Codex plugin marketplace for `tuolinagent`.

## 安装

打开 Codex，切换到 Plan 模式，复制下面这段话：

```text
请从这个 Codex 插件市场仓库安装 tuolinagent：
https://github.com/htshepherd/tuolinagent-marketplace

请使用下面两条命令：
codex plugin marketplace add htshepherd/tuolinagent-marketplace
codex plugin add tuolinagent@tuolinagent-marketplace

请只使用上面的安装地址和命令。

安装前先检查我的 Windows 电脑依赖。请运行 tuolinagent 插件里的 scripts/windows_check_dependencies.ps1。

如果缺少 Git、Python 3.10+、ffmpeg 或 MinerU，请先告诉我缺少什么、为什么需要、准备执行什么安装命令。

在我回复“同意安装依赖”之前，不要安装任何软件。
```

## 配置本地资料

把路径换成你的真实 raw 路径：

```text
请在当前项目里创建 config/tuolin-kb.config.json。
我的 raw 目录是 D:\LocalKnowledge\raw，请在配置里写成 D:/LocalKnowledge/raw。
保持 codex_adapter 模式，不要要求 Graphify API key。
不要移动、删除或重命名我的 raw 目录。
```

## 使用

整理知识库：

```text
整理一下拓霖知识库
```

按推荐继续：

```text
按推荐顺序继续整理
```

更新知识库：

```text
更新拓霖知识库
```

问答测试：

```text
用拓霖知识库回答：石英纤维隔热带适合哪些场景？
```
